## 📌 Project Overview

This project is a **low-fidelity wireframe** for an e-commerce homepage.  
The goal was to design a **clean, logical layout** that reduces friction and guides users clearly.

---

## 🧩 Sections Included

- ✅ Navigation Bar (Logo, Search, Cart, Login)
- ✅ Hero Section (Promo banner + CTA button)
- ✅ Promo Boxes (Offer, New Arrivals, Flash Sale)
- ✅ Product Grid (4 products with prices + Add to Cart)
- ✅ View All Products CTA
- ✅ Footer (About, Contact, Privacy, Social)

## 🎯 Key Principles Applied

- **Only Black, White, and Grey** (no colors or images)
- **Clear Call‑to‑Action buttons** (Shop Now, Add to Cart)
- **Logical hierarchy** (user knows where to look first)
- **Clean spacing** (reduces cognitive load)

## 🏆 What I Learned

- Wireframing is about **structure**, not style.
- Good UX removes **friction** and makes the user's journey smooth.
- Simple black‑and‑white layouts help focus on **flow and hierarchy**.
